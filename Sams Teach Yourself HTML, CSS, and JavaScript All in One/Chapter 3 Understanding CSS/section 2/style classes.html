Style Classes

A style class is a custom formatted text that you can apply to any element in a web page. First, let's wrap up some terminology:

  • Selector: h1, p, span
  • Property: font, color
  • Value: Courier, blue

With classes, you can apply different styles to selectors througout your file:

On the CSS:

  1. li.silly {font: 12pt Comic Sans MS;}
  2. li.serious {font: 12pt Arial;}
  3. p.subheader{
    font-weight: bold;
    color: #593d87;}

On the HTML:

  1. <li class="silly">Marvin's Munchies Inc.</li>
    • Marvin's Munchies Inc
  2. <li class="serious">MMI Investor Information</li>
    • MMI Inverstor Information
  3. <li class="subheader">subheader</li>
    • subheader
