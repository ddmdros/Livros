Basic Style Sheet

Layout Properties

Controls the physical positioning of content on a web page

  • block
    • Other: table, flex, grid
  • list-item
  • inline
    • inline-table, inline-flex, inline-grid
  • in-line-block
  • none
  • Widht, height
    • in - inches
    • cm - centimeters
    • em - ems
    • mm - millimeters
    • % percentage
    • px - pixels
    • pt - points
    • rem - rems

Tip: be consistent with your style properties! For example: stick with rems for font properties, and percentages for dimensions.

Formatting Properties

Controls the appearance of content on a web page

  • Border: establishes the visible boundary around an element with a box or partial box
    • border-width: the widht of the border edge
      • Often expressed in pixels: 5px, for instance
    • border-color: the color of the border edge
      • blue
    • border-style
      • solid: single line border
      • double: double-line border
      • dashed: dashed border
      • dotted: dotted border
      • groove: border with a groove appearance
      • ridge: border with a ridge appearance
      • inset: border with an inset appearance
      • outset: border with an outset appearance
      • none: no border (default)
      • hidden: the same as none
    • Properties that allows to set each side of the border individually:
      • border-left: the left side of the border
      • border-right
      • border-top: the top of the border
      • border-bottom
    • border: all the border sides
    • Radius: takes one or two values (horizontal radius, vertical radius) or one (used for both)
      • border-top-left-radius
      • border-top-right-radius
      • border-bottom-left-radius
      • border-bottom-right-radius
      • border-radius: 1rem
      • color: sets the inner region color of an element
      • background-color: sets the inner region background of an element
  • Alligment
    • text-alling: applies to the contnet of a block-level element, not to the element itself
      • center
      • justify
      • left
      • right
    • text-indent:
      • 12px
    • padding
      • padding-left
      • padding-right
      • padding-top
      • padding-bottom
  • Font
    • font-family: provides a prioritized list of font families
      • Arial, "Gill Sans", sans-serif;
    • font-size
    • font-style
      • normal
      • italic
    • font-weight
      • normal
      • lighter
      • bold
      • bolder
