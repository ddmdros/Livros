Aula 02: Tags

Tags

Vamos testar algumas formatações em texto.

Podemos usar algumas tags para modificar o texto:

  • Negrito: <p>conteúdo</p>
  • Itálico: <i>conteúdo</i>
  • Strong: <strong>conteúdo</strong>
  • Negrito e itálico: <b><i>conteúdo</b></i>
  • Sobrescrito: <sup>conteúdo</sup>
  • Subescrito: <sub>conteúdo</sub>
  • Texto inserido: <ins>conteúdo</ins>
  • Texto apagado: <del>conteúdo</del>
  • Itálico e um negrito no meio: <i>conteúdo geral<b>conteúdo negrito</b>conteúdo geral</i>
